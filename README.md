# cheatsheets

A collection of concise programming language and framework cheatsheets maintained by **xOpihuchus**.

## Overview

This repo contains quick-reference guides for many popular programming languages, frameworks, and technologies, including:

- C
- C++
- Java
- Go
- Rust
- TypeScript
- JavaScript
- Python
- Kotlin
- Spring Boot
- Design Patterns (C++, Java, Rust)

Each cheatsheet covers syntax, core concepts, design patterns, and practical usage for fast lookup and learning.

Browse the language- or framework-specific `.md` files in this directory for the cheatsheet you need.  
Great for study, interviews, or daily coding.
